## Simple Package Names

### Problem Snapshot
Small problem review. Customer sends photos, notes, or screenshots. Highway 38 sends back a clear written recommendation.

### Project Packet
Full project plan. Includes layout ideas, material list, build order, notes, and next steps.

### Shop Flow Review
Garage/shop/workspace improvement. Includes tool placement, workflow, storage, safety/access notes, and layout suggestions.

### Business Cleanup Packet
Small business organization. Includes service menu, quote sheet, customer intake form, tracker, and follow-up system.

### Digital Setup Packet
Website/AI/form setup. Includes website outline, page copy, AI prompts, form layout, and basic automation plan.

### Custom Work Build
Scoped custom output when the standard packages are close but not exact.

## GitHub Pages Setup

To publish the site:

1. Go to **Settings → Pages**
2. Under **Build and deployment**, select **Deploy from a branch**
3. Branch: `main` — Folder: `/ (root)`
4. Click **Save**

The site will be live at:

`https://rkrueth-maker.github.io/ForgeIQ/`

The `.nojekyll` file at the root disables Jekyll processing so the plain HTML/CSS site loads correctly.
