# Highway 38 Solutions sample image asset prompt

Use this prompt in another ChatGPT/image tool, stock-photo search, or design tool to replace the current SVG fallback visuals with better real-looking pictures.

## Rules

- Do not use copyrighted random web photos unless the license allows commercial website use.
- Best options: Rick's own photos, approved customer photos, licensed stock photos, or AI-generated images created for Highway 38 Solutions.
- Keep images practical, not glossy corporate stock.
- No people need to be identifiable.
- Use neutral shop/garage lighting, realistic clutter, tools, notes, and work surfaces.
- Export as JPG or WebP, around 1600px wide.
- Name files clearly and store under `/assets/`.

## Image set needed

### 1. Problem Snapshot / Garage bay
Create a realistic before/after style image for a cluttered garage bay becoming a clear vehicle lane with wall storage zones. Northern Minnesota home garage feel. Practical tools, bins, shelves, concrete floor, no luxury showroom.

Suggested filename:
`assets/photo-garage-snapshot.webp`

### 2. Project Packet / Garage layout
Create a realistic planning image showing a 28x40 garage concept: two overhead doors, parking lane, workbench zone, wall storage, material checklist, tape marks or sketch overlay. It should look like a practical project planning handoff, not a final architectural blueprint.

Suggested filename:
`assets/photo-project-packet.webp`

### 3. Shop / CNC Concept
Create a realistic small shop or CNC work area image showing a machine, material rack, part staging, operator travel path, and a simple concept overlay for infeed/machine/outfeed. Do not imply final robot guarding or certified design.

Suggested filename:
`assets/photo-cnc-concept.webp`

### 4. Business Cleanup
Create a realistic desktop/laptop workspace showing quote notes, a lead tracker spreadsheet, follow-up checklist, and service menu. Small contractor/service business feel. No visible private customer data.

Suggested filename:
`assets/photo-business-cleanup.webp`

### 5. AI Workflow Snapshot
Create a realistic dashboard/workflow setup image showing forms, spreadsheet tracker, prompt/template notes, and an owner dashboard. Small business AI workflow, not enterprise SaaS.

Suggested filename:
`assets/photo-ai-workflow.webp`

## Replacement instruction for web editor

After images are created and uploaded to `/assets/`, update `sample-library.html`, `index.html`, `real-proof-lab.html`, and `examples.html` to use the new photo files as primary images. Keep the existing SVGs as fallback links or archive assets.
