# AI Workflow Snapshot Sample

Purpose: show how a repeated business task becomes a small AI-supported workflow.

Customer sends:
- current process notes
- screenshots
- repeated tasks
- tools used
- desired outcome

Output:
- workflow map
- AI use-case list
- prompt starter
- tool-stack suggestion
- setup steps
- boundary notes

Example first AI use: draft follow-up messages from a lead tracker instead of rewriting the same message every time.
