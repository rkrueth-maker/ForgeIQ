Order flow

From messy request to delivered output.

This is the path to use for every real job.

1. Lead IntakeRaw request from form, email, text, or referral.
2. TriagePick desk, package, missing info, and next reply.
3. OrdersCreate order only when ready to quote/build.
4. Build SheetUse project, shop, business, cleanup, digital, custom, or Shop/CNC builder.
5. QACheck goal, inputs, assumptions, scope, boundary, links, and next action.
6. HandoffSend deliverable, first action, missing info, and optional upgrade.
