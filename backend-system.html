Order flow

From customer request to finished product.

This is the backend path every order follows.

1. IntakeForm captures customer, package, goal, messy details, files, budget, and timeline.
2. Lead rowRequest is entered into Leads with status, priority, assigned backend tab, and next action.
3. Build sheetUse Basic Layout Builder or Product Build Sheets to structure the work.
4. QA handoffCheck goal, assumptions, missing info, scope, links, and customer next action.
5. DeliverySend the finished output, folder link, next step, and relevant optional upgrade.
